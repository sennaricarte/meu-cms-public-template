Exemplo de página estática

Modelo de página em Markdown dentro de src/content/pages. Substitui este texto por um resumo com pelo menos cinquenta caracteres para SEO.

Conteúdo

Este ficheiro demonstra como usar Markdown (ou .mdx) para páginas institucionais: políticas, termos, FAQs, etc.

A URL pública corresponde ao caminho do ficheiro: /exemplo-pagina. Para subpastas, por exemplo legal/privacidade.md, a URL será /legal/privacidade.

Podes remover este exemplo quando criares as tuas páginas reais.
